## 铃之椅

欢迎来到铃之椅! 这是一个即时通讯项目, 为通讯提供更多的选择, 为人民服务

> [!NOTE]
> 本项目仍在实验阶段, [点我](final.md)可查看进展
>  
> 如果有任何问题，欢迎你提出来，我会不定时查看
>  
> 另外 Android 客户端也在开发, 但进展缓慢

### 使用

服务端:

  1. 克隆本仓库源代码到本地

  2. 运行 run.sh 或 run.bat

网页端:

  * 直接使用和服务端集成的网页 (推荐)

  * 克隆本仓库到本地并运行本地 HTTP 服务端
 
  * 静态网页 (不推荐, 有跨域问题)
